# スキルツリーデータパック

Minecraft Java Edition | スキル機能を追加するデータパックです。攻撃することなど特定のイベントでレベルなどが上がり、エフェクトやアイテムが付与されます。バニラの要素を崩さなく、最低限の機能のみを実装します。

## 動作確認済み環境

- [x] Minecraft Java Edition / v1.21
